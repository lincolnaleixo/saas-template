# Developer Guide

## Getting Started

### Prerequisites
- Docker installed
- Git configured
- Claude Code CLI installed

### Initial Setup

* Development: `./scripts/dev.sh` with hot reload and log tailing
* Production: `./scripts/prod.sh` with optimized builds and SSL/nginx
